# Design Assets Package

This directory contains all the design assets for the Astro360 application.

## Directory Structure

### `/icons`
Contains SVG icons used in the application.
- `*.svg`: Scalable Vector Graphics for UI icons.
- **Note**: The application primarily uses `lucide-react` for icons, but custom icons should be stored here.

### `/illustrations`
Contains high-fidelity illustrations and graphic elements.
- `*.png`: Raster images for hero sections and empty states.
- `*.svg`: Vector illustrations for scalability.

### `/animations`
Contains animation files.
- `*.json`: Lottie animation files for complex interactions.

### `/source_files`
Contains the source design files for the UI/UX.
- `*.fig`: Figma design files.
- `*.sketch`: Sketch design files.
- `*.pdf`: PDF exports of the design system and mockups.

## Usage
Developers should import assets from this directory into the `frontend/src/assets` folder as needed for implementation.
